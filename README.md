# 🚀 Task Manager App (TaskFlow)

A modern, production-style task management application built using **React, TypeScript, Redux Toolkit, and Tailwind CSS**.

It simulates a real-world productivity SaaS tool similar to Trello / Jira (frontend version).

---

## ✨ Features

- 📌 Create, edit, delete tasks
- 📊 Task status workflow (Todo → In Progress → Done)
- 🎯 Priority system (Low / Medium / High)
- 🔍 Filter tasks by status
- 💾 Persistent storage (localStorage)
- 🌙 Dark mode support
- ⚡ Optimized Redux state management
- 📱 Fully responsive UI

---

## 🛠 Tech Stack

- React 19
- TypeScript
- Redux Toolkit
- React Redux
- Vite
- Tailwind CSS

---

## 🧠 Architecture Highlights

- Feature-based folder structure
- Centralized Redux state management
- Memoized selectors (performance optimized)
- Reusable UI components
- Persistent state layer (localStorage sync)
